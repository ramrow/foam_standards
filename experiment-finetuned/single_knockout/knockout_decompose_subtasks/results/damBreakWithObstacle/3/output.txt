## Error 2: `log.interFoam` — Cannot find patchField entry for walls

**Diagnosis**: The `p_rgh` file in the `0/` directory is missing the `walls` boundary patch entry. The `blockMeshDict` defines a `walls` patch, but the `p_rgh` file only has `leftWall`, `rightWall`, `lowerWall`, `atmosphere`, and `defaultFaces`. The solver looks for a `walls` entry in `p_rgh` and cannot find it.

**Resolution**: Add the `walls` boundary patch entry to `0/p_rgh`. The `walls` patch should use `fixedFluxPressure` type with `value uniform 0;`. The file should have:

```
boundaryField

    }
    leftWall
    {

        value           uniform 0;

        value           uniform 0;
    }
    lowerWall
    {
        type            fixedFluxPressure;
        value           uniform 0;
    }
    atmosphere
    {
        type            totalPressure;
        p0              uniform 0;
        value           uniform 0;
    }
    defaultFaces
    {
        type            empty;
    }
}
```

## Error 3: `log.topoSet` — Attempt to return dictionary entry as a primitive

```

    }
    name            obstacle;
}
```
