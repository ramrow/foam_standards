In the `fvSolution` file, the preconditioner for the `h` solver is set to `DILU`:

```

    ...

```

The `h` equation involves a Laplacian solve, which produces a symmetric matrix system. When PIMPLE iterates with multiple outer correctors, the pressure-like correction step solves for `h` using a symmetric matrix, making `DILU` invalid.

```
    h

        relTol          0;
    }
}
```
