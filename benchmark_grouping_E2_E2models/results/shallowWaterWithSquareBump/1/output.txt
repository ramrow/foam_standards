Change:
```
h
{
    solver          PBiCGStab;
    preconditioner  DILU;
    tolerance       1e-12;
    relTol          0.01;
}
```

to:
```
h
{
    solver          PBiCGStab;
    preconditioner  DIC;
    tolerance       1e-12;
    relTol          0.01;
}
```

Keep the `hU` solver unchanged with `DILU` since its matrix is asymmetric. Similarly, `hUFinal` inherits from `$hU` and already uses `DILU`, which is fine for the asymmetric momentum solve.

This is consistent with the squareBump tutorial case, where the pressure-like equation solver for `h` uses `DIC` (the symmetric counterpart of `DILU`).
