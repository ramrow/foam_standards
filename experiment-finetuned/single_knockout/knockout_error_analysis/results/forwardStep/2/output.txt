   - Keep the `0/` directory contents exactly as they appear in the tutorial (U, T, p).

   ```bash
   blockMesh

   ```bash
   ls system/polyMesh/points
   ```  
   If the file is present, the mesh is ready for the solver.

   ```bash
   rhoCentralFoam
