## Error 1: `log.subsetMesh` — Invalid patch names

**Error:** `Valid patches are leftWall, rightWall, lowerWall, upperWall, frontWall, backWall` but the mesh has 6 patches. The `subsetMesh` command is being called with a cellSet that doesn't match any of these patch names.

**Root Cause:** The `topoSetDict` creates a cellSet named `obstacleCell`, but `subsetMesh` is looking for a cellSet with that name to use as a patch filter. The `subsetMesh` command syntax is `subsetMesh -overwrite -patch obstacleCell obstacleCell <cellSetName>`. Since `obstacleCell` is a cellSet (not a patch), this fails.

**Solution:** Change the `topoSetDict` to create a patch instead of a cellSet. Replace all `type cellSet` entries with `type patch`. The `topoSetDict` should look like:

```

## Error 1: `log.subsetMesh` — Invalid patch names

**Error:** `Valid patches are leftWall, rightWall, lowerWall, upperWall, frontWall, backWall` but the mesh has 6 patches. The `subsetMesh` command is being called with a cellSet that doesn't match any of these patch names.

**Root Cause:** The `topoSetDict` creates a cellSet named `obstacleCell`, but `subsetMesh` is looking for a cellSet with that name to use as a patch filter. The `subsetMesh` command syntax is `subsetMesh -overwrite -patch obstacleCell obstacleCell <cellSetName>`. Since `obstacleCell` is a cellSet (not a patch), this fails.

**Solution:** Change the `topoSetDict` to create a patch instead of a cellSet. Replace all `type cellSet` entries with `type patch`. The `topoSetDict` should look like:

```

(
