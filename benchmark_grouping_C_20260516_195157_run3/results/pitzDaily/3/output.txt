The syntax `(-20.6 0 -0.5)    v0` is the **old-style** vertex definition format. In modern OpenFOAM (Foundation versions), vertices are defined using only the coordinate syntax without explicit labels:

```
vertices
(

    (0 0 -0.5)
    (0 25.4 -0.5)
    (20.6 0 -0.5)
    (-20.6 0 0.5)
    (0 0 0.5)
    (0 25.4 0.5)
    (20.6 0 0.5)
);
```

The old style with `v0`, `v1`, etc., is still supported in some older/community versions but fails in others.
